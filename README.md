Automated deployment of fresh Moodle LMS installation on AWS Malaysia region using Terraform.

## Features
- Fresh Moodle installation from official Git repository
- Ubuntu 22.04 LTS base image
- Automated LAMP stack setup
- AWS Malaysia (ap-southeast-5) region deployment
- Security groups with HTTP/HTTPS/SSH access

## Prerequisites
- AWS account with access credentials
- Terraform >= 1.5 installed
- AWS CLI configured (optional)
